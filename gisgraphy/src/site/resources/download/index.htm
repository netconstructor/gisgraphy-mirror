Important notes

Here are the release of Gisgraphy. Sources distribution include the release and the sources (useful for developers or curious). binary distribution and sources distribution includes
  • fulltext and gisgraphy webapp
  • JETTY 's utilities to easily run gisgraphy
  • HTML documentation
  • SQL scripts
  • Config files

The release are designed to be easily run (you just need to install a PostgresQL Database and execute the command java -jar start.jar). To learn more on how to install Gisgraphy : see the installation guide
The release have been tested on Linux and Windows, on JVM 1.5 and 1.6. if you've got problems, consult and post a message on the forum
All the versions are available in ZIP (useful for Windows users) and tar.gz (useful for linux users) formats
