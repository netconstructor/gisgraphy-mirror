Gisgraphy client librairies

Some client librairies have been developped or are currently in progress :

OpenGTS

Gisgraphy is supported by OpenGTS. That means that you can do geocoding on OpenGTS via Gisgraphy. (see doc)

Java

The java client is currently in progress in a separate project on github. the developpement seems stoped, but a java client will probably be available is V 3.0. due to the fact that it is not developed by me, i will not support it.

Javascript

Some javascript utilities are availables. services.gisgraphy.com use it.

The code is available on the svn, in the distribution, or on the Gisgraphy site

Python

A client librairy is available. The code is available on the SVN, in the distribution Some other contribution are also be done by the community : https://github.com/JeffAMcGee/localcrawl/blob/master/gisgraphy.py
